# CRO Audit Checklist

Comprehensive checklist for evaluating landing page conversion rate optimization across all 10 scoring dimensions.

## 1. Above-the-fold Clarity

- [ ] Headline clearly states what the product/service does
- [ ] Headline focuses on visitor benefit, not company features
- [ ] Subheadline expands on the headline with supporting detail
- [ ] Visitor can understand the value proposition within 5 seconds
- [ ] Hero image or visual reinforces the message (not generic stock)
- [ ] No competing messages or distractions above the fold
- [ ] The page passes the "squint test" - key elements are visible when squinting

## 2. CTA Visibility and Copy Strength

- [ ] Primary CTA is visible above the fold without scrolling
- [ ] CTA button uses high-contrast color that stands out from the page
- [ ] CTA text is action-oriented and specific (e.g., "Start Free Trial" not "Submit")
- [ ] CTA copy communicates what happens after clicking
- [ ] Primary CTA is repeated at logical intervals down the page
- [ ] Secondary CTAs do not compete with or overshadow the primary CTA
- [ ] CTA button is large enough to tap easily on mobile
- [ ] Microcopy near CTA reduces anxiety (e.g., "No credit card required")

## 3. Social Proof Presence and Quality

- [ ] Customer testimonials include real names and photos
- [ ] Testimonials include specific outcomes or results (numbers, percentages)
- [ ] Recognizable company logos are displayed for B2B products
- [ ] Customer count or usage statistics are mentioned (e.g., "10,000+ teams")
- [ ] Case study links or summaries are available
- [ ] Star ratings or review scores from third-party platforms are shown
- [ ] Social proof is placed near decision points (near CTAs, near pricing)
- [ ] Video testimonials are included when possible

## 4. Trust Signals

- [ ] Security badges are displayed (SSL, SOC 2, GDPR compliance)
- [ ] Payment security indicators are visible near checkout/pricing
- [ ] Partner or integration logos are shown
- [ ] Industry certifications or awards are displayed
- [ ] Privacy policy link is accessible
- [ ] Money-back guarantee or free trial is prominently stated
- [ ] Company contact information is available (address, phone, email)
- [ ] Media mentions ("As seen in...") are included if applicable

## 5. Form Simplicity

- [ ] Form has the minimum number of fields necessary
- [ ] Each field has a clear, visible label (not just placeholder text)
- [ ] Required vs optional fields are clearly distinguished
- [ ] Form uses appropriate input types (email, phone, dropdown vs text)
- [ ] Error messages are inline and descriptive
- [ ] Form is single-step or has a clear progress indicator for multi-step
- [ ] Smart defaults are used where possible
- [ ] Social login or SSO options are available to reduce friction
- [ ] Form button text matches the expected action

## 6. Page Speed and Mobile Responsiveness

- [ ] Page loads in under 3 seconds on desktop
- [ ] Page loads in under 4 seconds on mobile (3G connection)
- [ ] Images are optimized and use modern formats (WebP, AVIF)
- [ ] Lazy loading is implemented for below-the-fold images
- [ ] No Cumulative Layout Shift (CLS) during load
- [ ] Page is fully responsive across common breakpoints
- [ ] Touch targets are at least 44x44px on mobile
- [ ] Text is readable without zooming on mobile
- [ ] No horizontal scrolling on any viewport size

## 7. Visual Hierarchy and Scan-ability

- [ ] Single H1 tag contains the primary headline
- [ ] Heading hierarchy follows logical order (H1, H2, H3)
- [ ] Content is broken into distinct, scannable sections
- [ ] Bullet points and short paragraphs are used instead of dense text
- [ ] Important information is bolded or visually emphasized
- [ ] Whitespace is used effectively between sections
- [ ] Visual flow guides the eye from headline to CTA
- [ ] Icons or illustrations support text content
- [ ] Content sections alternate layout (text-left/image-right, then reverse) for variety

## 8. Objection Handling

- [ ] FAQ section addresses common questions and concerns
- [ ] Pricing is transparent (or clearly stated as "contact for pricing")
- [ ] Comparison table shows advantages over alternatives
- [ ] Money-back guarantee or risk reversal is stated
- [ ] "How it works" section reduces complexity objections
- [ ] Integration or compatibility concerns are addressed
- [ ] Support availability is mentioned (live chat, email, phone)
- [ ] Migration or onboarding assistance is offered
- [ ] Data security and privacy concerns are addressed

## 9. Urgency and Scarcity (Authentic)

- [ ] Any time-limited offers have real, verifiable deadlines
- [ ] Limited availability claims are genuine and specific
- [ ] Countdown timers reflect actual deadlines (not fake recurring timers)
- [ ] Early-bird or launch pricing has clear expiration
- [ ] Waitlist or limited beta access creates authentic scarcity
- [ ] Seasonal or event-tied promotions are time-relevant
- [ ] No dark patterns - urgency feels helpful, not manipulative
- [ ] Loss aversion is addressed (what they miss by not acting)

## 10. Copy Clarity and Benefit-driven Language

- [ ] Headlines lead with benefits, not features
- [ ] Each feature mentioned is paired with a benefit ("so you can...")
- [ ] Copy uses the visitor's language (not internal company jargon)
- [ ] Sentences are short and direct (aim for 8th-grade reading level)
- [ ] Specific numbers and outcomes replace vague claims
- [ ] Active voice is used throughout
- [ ] Copy addresses the visitor directly using "you" and "your"
- [ ] Emotional and rational benefits are both represented
- [ ] The unique value proposition is clear - why this over alternatives

## Quick-win Checklist (High Impact, Low Effort)

These items typically produce the fastest conversion lifts:

1. [ ] Rewrite CTA from generic ("Submit") to specific ("Start My Free Trial")
2. [ ] Add microcopy under CTA ("No credit card required - 14 day free trial")
3. [ ] Add 3 customer logos above the fold
4. [ ] Reduce form fields to the absolute minimum
5. [ ] Add a testimonial with a specific result near the primary CTA
6. [ ] Make the headline benefit-focused instead of feature-focused
7. [ ] Add a trust badge near the form or CTA
8. [ ] Remove navigation links that lead away from the conversion goal
9. [ ] Increase CTA button size and contrast
10. [ ] Add a clear "How it works" section (3 simple steps)
