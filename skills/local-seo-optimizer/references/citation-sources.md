## NAP Consistency Guidelines

### Formatting Standards
- **Business name:** Use exact legal/signage name consistently everywhere. Do not add keywords, city names, or taglines.
- **Address:** Use the USPS-standardized format. Be consistent with abbreviations (St vs Street, Ste vs Suite).
- **Phone:** Use the same phone number everywhere. Pick either local or toll-free and stick with it. Format: (555) 555-5555.
- **Website:** Use the same URL format everywhere (with or without www, with or without trailing slash - just be consistent).

### Common Consistency Issues
1. **Name variations:** "Joe's Pizza" vs "Joes Pizza" vs "Joe's Pizza LLC"
2. **Address abbreviations:** "123 Main Street Suite 4" vs "123 Main St Ste 4"
3. **Phone format:** "(555) 555-5555" vs "555-555-5555" vs "5555555555"
4. **Old addresses:** Previous locations still listed on some directories
5. **Duplicate listings:** Multiple listings for the same business on one platform

### How to Fix Inconsistencies
1. Decide on one standard format for name, address, and phone
2. Create a spreadsheet tracking all known citations
3. Update each listing to match the standard format
4. Remove or merge duplicate listings
5. Set calendar reminders to re-audit quarterly

### Monitoring
- Check citations quarterly for accuracy
- Monitor for unauthorized changes or duplicates
- After any address, phone, or name change, update all citations within one week
- Track which citations are indexed by Google using site: searches
